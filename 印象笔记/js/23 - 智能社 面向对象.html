18-智能社 面向对象

作者： 2067659951@qq.com
面向对象：不了解原理，会使用功能  
比如几千号人的大公司，CEO找设计部、技术部、采购部、销售部安排开发新产品的工作，不需要关注部门怎么工作的，只需要部门完成产品。

使用面向对象思想包括两种人：
1、用对象的人（之前扮演的 Array、Date、Math）
2、写对象的人（之后扮演的）

面向对象编程(OOP)的特点：

1、抽象 抽 ——> 把跟问题相关的、最主要的特征抽出来
比如员工管理系统发工资 关注员工的姓名、工号、职位、工资最主要的特征 相亲系统 关注身高、体重

2、封装 看不到里面的东西，用好表面的功能就行了
把对象内部的结构和细节封起来，用对象的人不需要关心，只有写对象的人需要关心

3、继承 从父类上继承一些方法、属性，子类又可以有自己的一些特性。

遗传（不考虑变异的情况）
A的功能 ——> B
可以保证最大地重用代码
拖拽 ——> 限制范围的拖拽(继承拖拽、再改一下)

----3.1  多重继承(很少用，会使程序混乱)：
    集装箱货车 继承两个类的 ——> 盒子类：能装东西 + 汽车类：有轮子、会跑
----3.2  多态(对强类型语言：Java、C++比较有用，对JS弱类型的语言意义不大)

什么是对象：黑盒子，不了解内部结构，知道表面的各种操作(按钮)
手机、电脑、电视都是对象，不需要了解电视的内部工作原理，只需要会用按钮：会调音量、调台。
Date对象的getFullYear()方法 返回给定日期或本地时间的一个四位数的年份数字。
Array对象的sort()方法 对数组的元素进行排序，并返回数组

            width: 200px;
            height: 200px;
            background-color: #ccc;
            display: none;
        }
    </style>
    <script>
       window.onload = function(){//function() 忘写括号
            var oTab = document.getElementById("tab");
            var aBtn = oTab.getElementsByTagName("input");
            var aDiv = oTab.getElementsByTagName("div");
            for(var i = 0; i < aBtn.length; i++){
              aBtn[i].index = i;

                   };
                   this.className = "active";

            }  
       }

面向过程到面向对象步骤：

1、

函数不能有嵌套的函数，把函数放到window.onload外面去,然后取个名，把名字放回去。
function fnClick(){
    for(var i = 0; i < aBtn.length; i++){
           aBtn[i].className = "";
                aDiv[i].style.display = "none";
    };
    this.className = "active";
    aDiv[this.index].style.display = "block";
};

aBtn.onclick = fnClick;

2、

运行检查错误，将fnClick()提示错误的变量变成全局变量，将变量名放回去。
var aBtn = null;
var aDiv = null;
aBtn = oTab.getElementsByTagName("input");
aDiv = oTab.getElementsByTagName("div");

window.onload                                         构造函数

并不是所有的选项卡都叫"tab"，为了通用起见，需要一个参数传递：function TabSwitch(id)

所以，所有的aBtn和aDiv前面都得加this.

function TabSwitch(id){

面向对象写法：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title></title>
    <style type="text/css">
        input{
            background-color: white;            
        }
        .active{
            background-color: orange;
        }
        #tab div{
            width: 200px;
            height: 200px;
            background-color: #ccc;
            display: none;
        }
    </style>
    <script>
       window.onload = function(){
            new TabSwitch("tab");
       }

       function TabSwitch(id){
            var _this = this;
            var oTab = document.getElementById(id);
            this.aBtn = oTab.getElementsByTagName("input");
            this.aDiv = oTab.getElementsByTagName("div");
            for(var i = 0; i < this.aBtn.length; i++){
              this.aBtn[i].index = i;    
              this.aBtn[i].onclick = function(){
                   _this.fnClick(this);
              };    
            }  
       }

       TabSwitch.prototype.fnClick = function (aBtn){
           for (var i = 0; i < this.aBtn.length; i++) {
               this.aBtn[i].className = "";
               this.aDiv[i].style.display = "none";
           }
           aBtn.className = "active";//this要表示当前点击的按钮，而这里如果直接用则是当前的对象，所以要改为aBtn
           this.aDiv[aBtn.index].style.display = "block";
       };        
    </script>
</head>
