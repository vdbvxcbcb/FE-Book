14-Web 字体

Web 字体

任务目标：学习如何将web字体应用到web页面，使用第三方服务，或者编写自己的代码。


使用font-family属性的字体栈来指定首选字体，后面是备选的Web安全字体，然后是默认的系统字体，但是为了确保每种字体都显示正常，这样增加了测试的开销。

一种更好的方案是使用Web 字体（即使在IE6中都有效），Web字体是一种CSS特性，可以指定访问网站时下载字体文件。

注意：

1、浏览器支持不同的字体格式，因此您需要多种字体格式才能获得良好的跨浏览器支持。例如,大多数现代浏览器都支持WOFF / WOFF2(Web Open Font Format versions 1 and 2，Web开放字体格式版本1和2)，它是最有效的格式，但是旧版本的IE只支持EOT (Embedded Open Type，嵌入式开放类型)的字体。旧版本的iPhone和Android浏览器需要一个包含SVG型式的字体来支持。

2、字体一般都不能免费使用。您必须为他们付费，或者遵循其他许可条件，比如在代码中(或者在站点上)提供字体创建者，不应该在没有授权的情况下偷用字体。

Web字体技术从 IE4 开始就得到支持了。IE9 只支持 .eot 类型的字体，IE中只有IE9支持@font-face规则。

@font-face规则指定名为"自己喜欢的名字"的字体，并指定在哪里可以找到它的URL。（生成一个全新的字体文件，再通过src属性url()方法外链这个字体文件，可以是绝对或相对URL。）

@font-face可以简化字体名称

@font-face {
  font-family: YH;
  src: local("Microsoft Yahei");
}

.font {
    font-family: YH;
}

@font-face {
  font-family: BASE;
  src: local("HelveticaNeue-Light"), local("Helvetica Neue Light"),  local("PingFang SC"), local("Microsoft YaHei"), local(sans-serif);
}
body {
    font-family: BASE;
}
.xxxx {
    font-family: BASE;
}


刀枪不入的@font-face规则兼容格式（即可直接应用font-family: 'YourWebFontName';）

@font-face {
        font-family: 'YourWebFontName';
        src: url('YourWebFontName.eot?') format('eot');/*IE*/
        src:url('YourWebFontName.woff') format('woff'), url('YourWebFontName.ttf') format('truetype');/*non-IE*/
}

@font-face {
        font-family: 'YourWebFontName';
        src: url('YourWebFontName.eot'); /* IE9 Compat Modes */
        src: url('YourWebFontName.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
             url('YourWebFontName.woff') format('woff'), /* Modern Browsers */
             url('YourWebFontName.ttf')  format('truetype'), /* Safari, Android, iOS */
             url('YourWebFontName.svg#YourWebFontName') format('svg'); /* Legacy iOS */
}

stylesheet.css 文件包含了需要的生成好的 @font-face 代码。把所有的 @font-face块复制到你的 css 文件中，修改url为正确路径即可使用。

CSS：

引入@font-face规则找到字体文件和对应的字体。

@font-face {
           font-family: 'playfair_displayregular';
           src: url('fonts/playfairdisplay-regular-webfont.woff2') format('woff2'),
url('fonts/playfairdisplay-regular-webfont.woff') format('woff');
           font-weight: normal;
           font-style: normal;
}

@font-face {
          font-family: 'open_sansregular';
          src: url('fonts/opensans-regular-webfont.woff2') format('woff2'),
url('fonts/opensans-regular-webfont.woff') format('woff');
          font-weight: normal;
          font-style: normal;
}

然后使用font-family属性应用

html {
  font-size: 10px;
  margin: 0;
  font-family: 'playfair_displayregular',sans-serif;
}

font-family 指定自己喜欢的任何名字为字体命名，只要你始终在CSS中使用该名称。

src 指定要导入的字体文件的路径。列出多个声明，用逗号分隔——浏览器会搜索并使用它能使用的第一个——因此，最好是把新的、更好的格式比如WOFF2放在前面，把偏老的，不是那么好的格式像TTF这样的放在后面。唯一的例外是EOT格式字体——放在首位是为了修复几个在旧版本IE中的bug。旧版本IE的bug：即使旧版本IE不能使用该字体，也会尝试使用找到的第一个字体。

font-weight/font-style  指定字体的粗细，以及是否斜体。如果您正在导入相同字体的多个粗细，您可以指定它们的粗细/样式，然后使用不同的font-weight/font-style来选择它们之间的不同值，而不必调用font family下不同名称的成员。
