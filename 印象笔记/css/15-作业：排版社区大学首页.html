排版社区大学首页

CSS：

* {
  box-sizing: border-box;
}

h1,h2{
  font-family:'open_sansregular',serif;
  letter-spacing: 2px;
}

p,li{
  line-height: 1.5;
  font-size: 1.6rem;
  letter-spacing: 0.5px;
  word-spacing: 3px;
}

a{
  color: #a66;
  outline: none;
}

a:hover,a:focus{
  text-decoration: none;
}

a:active{
  color: #AA4C4C;
  text-shadow: 0px 0px 1px #AA4C4C;
}

a[href*="http"]{
  padding-right: 20px;
  background: url('external-link-52.png') no-repeat 100% 0;
  background-size: 16px 16px;
}

nav ul{
  padding-left: 0; /*注意：整个<ul>（包括左边的点）往左移，<li>的宽度变大为下面的width:100%做铺垫*/
  margin-top: 0.8rem;/*把整个<ul>（包括左边的点）往上移*/
}

nav li {
  list-style-type: none;/*必需：去掉影响外观的点*/
  margin-bottom: 2rem;/*关键点一调整li之间的距离，分隔开*/
}

nav li a {
  text-decoration: none;/*必需：去掉影响外观的下划线*/
  display: inline-block;/*关键点二：<a>为行内元素所以设置为inline-block，其实设置block也没关系，因为inline-block好用而习惯偏好用inline-block*/
  width: 100%;/*关键点三：使原本每个<a>仅有文字宽的宽度变得可以铺满对应的<li>宽度*/
  line-height: 3;/*关键点四：起到上下加高+文字垂直居中的作用，利用行高撑开父容器的高度*/
  text-align: center;/*关键点五：文字水平居中，因为<a>文字被拉宽，所以需要水平+垂直居中。*/
  font-size: 2.5rem;/*改变文字大小，行高会相应改变*/
  border: 1px solid #a66;/*添加边框*/
}

nav li a:focus, nav li a:hover {
  color: white;
  background: #a66;
}

nav li a:active {
  color: white;
  background: black;
}

<ul>设置padding-left为0，因为整个<ul>不再包括左边的点，但是<ul>继续遵循元素的7个水平属性的总和等于父元素的width，所以宽度往右边弥补了，<ul>往左移并且宽高不变，<li>宽度变大。

<nav>的蓝色部分为<ul>的总宽度
