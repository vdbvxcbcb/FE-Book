什么是 HTML?

HTML (HyperText Markup Language) 指的是超文本标记语言，一种用于定义网页内容结构的标记语言，用于告诉您的浏览器如何构造您访问的网页。

超文本：超文本包含很多元素，比如文字、图片、链接到其他超文本的链接（超链接）等。超文本的基本特征就是可以超链接文档。超文本的一种表现形式就是网页。

HTML 标记标签通常被称为 HTML 标签 (HTML tag)。

HTML标签不区分大小写，最好使用小写字母。

HTML 文档也被称为网页。


剖析一个 HTML 元素

元素：


开始标记 + 内容 + 结束标记 = 元素

开始标记 + 结束标记 = 匹配标记

标记（tag 标签）

嵌套元素：

嵌套元素：嵌套——元素放到其它元素之中。（即两道门。进去：先开一道门再开另一道门。出来：先关一道门再关另一道门。）


块级元素和行内元素：

设计一个页面时，一般都是先从较大的块开始（块级元素），在完善页面时才加入行内元素。

这些元素的类型是通过文档类型定义（DTD）来指明。block 或 inline 是 CSS 术语。

块级：display:block;
行内：display:inline;

块级元素（独占一行）：前后各有一个换行，占满一行。
块级元素大多为结构性标签，用于展示页面上结构化的内容：例如段落、列表、导航菜单、页脚等等。
  1. 总是在新行上开始；
  2. 宽高、行高、内外边距都可以改变；
  3. 宽度默认100%，除非设定一个宽度；
  4. 块级元素中可以包含块级元素和行内元素。
                  
行内元素：行内元素大多为描述性标签。
  1.  和其他元素都在一行上；
  2. 宽高是内容（文字或图片）的宽高。
  3. 宽高和内外边距都不可以改变；
  4. 行内元素只包含行内元素，不包含块级元素。

行内元素，需要注意如下：
  • 设置宽度width 无效。
  • 设置高度height 无效，可以通过line-height来设置。
  • 设置margin 只有左右margin有效，上下无效。
  • 设置padding 只有左右padding有效，上下无效。注意元素范围是增大了，但是对元素周围的内容是没影响的。

<em> 是一个内联元素，<p>是一个块级元素。每个段落间都有一些间隔：因为默认的浏览器有着默认的展示<p>元素的 CSS 样式。

<em>first</em><em>second</em><em>third</em>

<p>fourth</p><p>fifth</p><p>sixth</p>





<a>属于行内非替换元素；<input>和<img>一样属于行内替换元素，替换元素拥有内在尺寸。

明确地来说，input是属于inline-block（CSS2.1 新增的值）元素，但inline-block 的概念最早是由替换元素演化而来的，

替换元素就是浏览器根据元素的标签和属性，来决定元素的具体显示内容。
例如浏览器会根据<img>标签的src属性的值来读取图片信息并显示出来；又例如根据<input>标签的type属性来决定是显示输入框，还是单选按钮等。
HTML中的<img>、<input>、<textarea>、<select>、<object>都是替换元素。这些元素往往没有实际的内容，即是一个空元素。

inline-block：可以设置宽高，也可以同行。

属性

属性包含元素的额外信息，这些信息不会出现在实际的内容中。属性以 名称/值 对的形式出现：属性名="属性值"。（或者说 属性= 属性名 + 属性值）


一个属性必须包含如下内容：
1. 在元素和属性之间有个空格 (如果有一个或多个已存在的属性，就与前一个属性之间有一个空格。)
2. 属性后面紧跟着一个“=”符号.
3. 有一个属性值,由一对引号“ ”引起来.


元素命名

应该根据“它们是什么”来为元素命名，而不是根据表现样式！

类应该用在概念相似的元素上，ID应该用于不同的唯一元素上，只有绝对确定只出现一次时使用！



布尔属性

没有值的属性。因为布尔属性只能有跟它的属性名一样的属性值<input type="text" disabled="disabled">，所以简写成没有值的属性：

<input type="text" disabled>
<input type="text">


添加引号，单引号和双引号不能在一个属性值里面混用（用成对的单引号或成对的双引号都可以，风格而已）。



分析HTML文档

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>My test page</title>
</head>
<body>
<p>This is my page</p>
</body>
</html>

<!DOCTYPE html>: 声明文档类型。类似于链接，它告诉 web 浏览器页面使用哪个 HTML 版本进行编写。 规定了HTML页面必须遵从的良好规则，能自动检测错误和其他有用的东西。

DOCTYPE有两种风格：严格（strict）和过渡（transitional），过渡版本允许使用废弃元素，严格版本禁止使用废弃元素。

常用的 DOCTYPE 声明：HTML 5——<!DOCTYPE html>

DOCTYPE作用：
● 分析页面使用哪个DTD（文档类型定义），进行文档有效性验证。
● 指定页面的使用标准和版本，是区分遗留文档和符合标准文档的手段。
● 决定使用哪种渲染模式。

渲染模式
● Quirks Mode (怪异模式)
● Almost standard Mode (接近标准模式)
● Standard Mode (标准模式)


XHTML是HTML的严格版本，遵守XML编码规定，例如属性必须包含引号，元素必须是封闭的，只包含一个错误就无法显示页面。

<html></html>: 是一个根元素。告诉浏览器文件的内容是HTML。

 <meta charset="utf-8">: 字符集编码。这个元素设置文档使用utf-8字符集编码，utf-8字符集包含人类大部分的文字，基本能识别你放上去的所有文本内容。毫无疑问要使用它，并且它能在以后避免很多其他问题。

<title></title>: 设置页面标题，出现在浏览器标签上，当你标记/收藏页面时可用来描述页面。

<body></body>:  页面主体，包含访问浏览器页面时显示在页面上的内容：文本，图片，音频，游戏等等。





HTML中的空白

下面的两个代码段是等价的：
<p>Dogs are silly.</p>

<p>Dogs are
silly.</p>

无论你用了多少空白(包括空白字符，包括换行), 连续出现的空白字符都会变为一个单独的空格符。

为什么使用那么多的空白呢? 答案是为了可读性 —— 格式化代码。eg:在我们的HTML代码中，我们让每一个嵌套的元素以两个空格缩进。你应该坚持使用某种风格格式化代码。





字符实体引用

如果你真的想要在文本中使用符号&或者小于号, 而不想让字符被浏览器视为代码并被解释?我们必须使用字符实体引用 —— 表示字符的特殊编码。

每个字符实体引用以符号&开始, 以分号 ; 结束。如需显示小于号，我们必须这样写：&lt; 或 &#60;

不间断空格（non-breaking space）：HTML 中的常用字符实体是不间断空格(&nbsp;)。


如需完整的实体符号参考，请访问我们的 HTML 实体符号参考手册。





HTML注释

注释被浏览器忽略，而且对用户不可见。

HTML注释的目的是允许你描述你的代码是如何工作的和不同部分的代码做了什么等等。

为了注释一段HTML中的内容，你需要将其用特殊的记号<!--和-->包括起来， 比如：

<p>I'm not inside a comment</p>

<!-- <p>I am!</p> -->


小结：没有内容的元素叫空元素。没有值的属性叫布尔属性。
